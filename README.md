# Sistema_escolar
Sistema escolar - projeto de Pi Fatec
O Sistema de Gestão de Ocorrências Escolares é uma plataforma web desenvolvida para auxiliar escolas na organização, registro e acompanhamento de ocorrências relacionadas aos alunos. Ele permite que professores, coordenadores e responsáveis (pais) interajam de forma centralizada e segura com as informações educacionais, oferecendo mais transparência, agilidade e controle institucional.

O sistema organiza dados, gera análises e facilita a comunicação entre escola e família, melhorando o acompanhamento do desempenho e comportamento dos alunos.
